# WeatherForecast
## Прогноз погоды.
#### Приложение для получения семидневного прогноза погоды из любой 
#### точки мира. Для выбора места используется карта мира, при нажатии на 
#### выбранную точку которой, происходит запрос прогноза и вывод его на следующем
#### экране. При нажатии на сегмент семидневного прогноза, появляется экран с подробным
#### прогнозом выбранного дня, а также прокрутка с часовым прогнозом на 48 часов.
